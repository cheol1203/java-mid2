## 💡 핵심 학습 내용

### 제네릭

제네릭은 클래스나 메서드에서 사용할 타입을 미리 고정하지 않고,  
사용 시점에 타입을 지정할 수 있도록 해주는 문법입니다.

이를 통해 불필요한 형변환을 줄이고, 컴파일 시점에 타입 오류를 확인할 수 있습니다.

### ArrayList

ArrayList는 배열을 기반으로 동작하는 List 구현체입니다.  
인덱스를 통한 조회가 빠르지만, 중간 삽입과 삭제가 많을 경우 성능이 떨어질 수 있습니다.

### LinkedList

LinkedList는 노드들이 서로 연결된 구조입니다.  
중간 삽입과 삭제에는 유리하지만, 특정 인덱스에 접근할 때는 순차 탐색이 필요합니다.

### List

List는 순서가 있고 중복을 허용하는 자료구조입니다.  
대표적인 구현체로는 ArrayList와 LinkedList가 있습니다.

### Hash

Hash는 데이터를 빠르게 저장하고 조회하기 위한 방식입니다.  
해시 코드를 사용하여 데이터를 저장할 위치를 결정합니다.

### Set

Set은 중복을 허용하지 않는 자료구조입니다.  
중복 제거가 필요할 때 자주 사용됩니다.

### HashSet

HashSet은 해시 기반의 Set 구현체입니다.  
데이터의 중복을 제거하고 빠르게 조회할 수 있습니다.

### Map

Map은 Key-Value 형태로 데이터를 저장하는 자료구조입니다.  
Key는 중복될 수 없고, Value는 중복될 수 있습니다.

### Stack과 Queue

Stack은 나중에 들어온 데이터가 먼저 나가는 LIFO 구조입니다.  
Queue는 먼저 들어온 데이터가 먼저 나가는 FIFO 구조입니다.

### 순회와 정렬

컬렉션에 저장된 데이터를 반복문, Iterator, 향상된 for문 등을 통해 순회할 수 있습니다.  
또한 정렬 기준을 지정하여 데이터를 원하는 순서로 정렬할 수 있습니다.
